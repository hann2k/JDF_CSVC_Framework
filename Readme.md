# JDF / CSVC Framework

LLM 위에서 동작하는 기준 기반 구조 분석 프레임워크

---

## 개요

입력된 질문을 기준 단위로 분해하고,  기준 간 충돌 및 책임 구조를 분석하여 결과를 출력합니다.

---

## 사용 방법

1. 새로운 세션을 열고, 모든 페르소나를 삭제합니다.
2. gtp: 최신프레임워크.pdf 파일을 그대로 업로드한 후, 질문을 입력하고 프롬프트를 보냅니다.
3. gemini: 최신 프레임워크.pdf 파일을 뷰어로 열어서, 전체 복사 후, 프롬프트에 붙여넣고 보냅니다. 이후 질문을 입력하고 프롬프트를 보냅니다.

프롬프트 예시는 아래를 참조하세요.

4. Gemini는 파일을 통째로 올리면 잘 작동하지 않습니다.
5. 질문까지 올렸으면 결과가 나올때까지 기다립니다. 

---

## 입력 예시

입력: 내용은 임의로 작성 가능합니다.
예)테슬라 오토파일럿 사고 분석<Enter>
예)국회의원 천하람이 이재명 대통령의 국무회의 발언(학생들 현장학습,운동회 못하는것에 대한)을 비난했다. 천하람은 교육현장의 현실을 전혀 모르고 기사 몇개 대충 읽고 자기 마음대로 판단한 무책임하고 위험한 현실 왜곡이라고 한다. 천하람의 이 발언에 담긴 의도를 분석하라. 천하람의 발언 전문은 페이스북 23시간전 글에서 확인할 수 있다.<Enter>

출력:
기준 분해 → 충돌 구조 → 책임 분석
