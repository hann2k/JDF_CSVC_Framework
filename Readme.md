# JDF / CSVC Framework

LLM 위에서 동작하는 기준 기반 구조 분석 프레임워크

---

## 개요

입력된 질문을 기준 단위로 분해하고,  
기준 간 충돌 및 책임 구조를 분석하여 결과를 출력합니다.

---

## 사용 방법

1. 문서를 LLM에 그대로 입력
2. 질문 입력
3. 구조 분석 결과 확인

---

## 예시

입력:
테슬라 오토파일럿 사고 분석

출력:
기준 분해 → 충돌 구조 → 책임 분석
